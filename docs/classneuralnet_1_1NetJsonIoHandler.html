Saves information about a network to a json file.

Exports Net to json file in human readable format. File will contain following data: size of input layer, number of layers, types of layers, their sizes, numbers of inputs and weight matrices.

Parameters
networkNet object which you want to export.
file_pathPath to location where you want the file to be created followed by a file name without extension suffix.
Exceptions
logic_errorIf network is empty.
runtime_errorIf file couldn't be created.

Loads network data from a json file.

Loads information about a network from json file and based on that data sets size of network input layer and adds appropriate layers and their weights. Types of layers handled by this funcion are: ReLuLayer, SoftmaxOutputLayer and SigmoidOutputLayer.

Parameters
networkNet object to which you want to load architecture from file.
file_pathPath to file containing network.
Exceptions
logic_errorIf network is not empty.
runtime_errorIf file couldn't be opened.
